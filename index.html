Frequently Asked Questions

These are some important FAQs every developers need to know.

Flexbox is for unidirectional dimensional layout (either row or column) whereas grid is for 2 dimensional layout. Number of columns or rows can be defined in grid but cannot be defined in flexbox.

The main difference between Bootstrap and Tailwind is that Bootstrap pre-built components that provide a responsive UI (user interface) kit. On the other hand, Tailwind comes with predesigned widgets that facilitate web developers to design a website from scratch. (Reference: askanydifference.com)

It is a box-like layout where margin, border, padding, and height-width are defined for an element in CSS styling.

A semantic tag is a tag with a meaningful name tag which is easier for the user to understand its purpose, and place in the body of the code. An example of semantic tag is nav in HTML5 which was menu in HTML4.
